2. Pengertian dari • Instuction Register • Program Counter • General Purpose Register • Memory Address Register • Memory Data Register pada proses register CPU


1.  program counter = register yang menunjukan ke instruksi berikutnya yg harus di ambil dan di jalankan.
2. instuction register   = registrasi yang menyimpan instruksi yg sedang di jalankan.
3. general purpose register = registrasi yg mempunyai kegunaan umum yg berhubungan dgn data yg diproses.
4. memory address register = registrasi yg digunakan untuk menampung data atau instruksi hasil pengiriman dari memori utama ke CPU  atau menampung data  yg akan direkam ke memori utama yg akan diambil dari hasil oleh pengolah CPU.
5. memory data register = digunakan untuk menampung alamat data atau instruksi pada memori utama yang akan diambil atau di letakan

20. jelaskan general purpose RISK​


Jawaban:

Register tidak dapat dilepaskan dari mikroprosessor, sebab pada mikroprosessor terdapat register yang berfungsi untuk menyimpan sementara hasil dari tahapan operasi arithmetika dan logika pada mikroprosessor. Register dalam bahasa rakitan menggunakan real mode memory yang sesuai dengan mikroprosessor Intel generasi 8088 s/d Pentium.

Register yang terdapat pada mikroprosessor Intel terdiri dari :

General purpose register (register serbaguna)

Pointer register (register pointer)

Index register (register indeks)

Segment register (register segment)

Flag register (register status).

Semua register di atas lebarnya 32 bit, kecuali register segment (CS, DS, ES, SS, FS dan GS) hanya 16 bit. Register 32 bit dapat digunakan sebagai register 16 bit, kecuali register General purpose register dapat dibagi menjadi 8 bit (AL,AH, BL, BH, CL, CH, DL dan DH) yang berasal dari 16 bit (AX, BX, CX, DX). Register 32 bit diberi kode di depan register dengan E misalnya: EAX, EBX, ECX dan EDX.

23. tuliskan 3 perbedaan antar general purpose computer dan special purpose computer


Special-purpose Computer adalah komputer yang digunakan untuk menangani satu jenis masalah khusus. 39. Tuliskan 3 perbedaan antara general purpose computer dan special purpose computer


1. Special Purpose Computer


Special-purpose computer digunakan untuk menyelesaikan pekerjaan ataupun aplikasi khusus.
Ciri-cirinya adalah :
1. Komputer yang dirancang untuk kebutuhan khusus
2. Program tertentu sudah tersimpan di dalam komputernya
3 . Dapat berupa komputer analog maupun digital
4. Umumnya adalah komputer analog

2. General Purpose Computer


General Purpose Computer merupakan komputer yang dibuat untuk keperluan secara umum dan digunakan setiap hari, sehingga komputer tersebut dapat digunakan untuk mengerjakan berbagai macam pekerjaan sesuai dengan kemampuan dan usernya.
Ciri-Cirinya adalah :
1. Komputer yang dirancang untuk menyelesaikan bermacam-macam masalah
2. Dapat berupa komputer analog maupun digital
3 . Umumnya adalah komputer digital

23. jelaskan tentang program pengolah kata notepad?​


Jawaban:

Notepad merupakan aplikasi yang dikemas satu paket bersama dengan sistem operasi Microsoft Windows, dan menjadi tanda stabil-nya sistem Windows. Notepad.exe terletak di dalam folder sistem Windows utama, yaitu di ‘C:/Windows/’ dengan nama file-nya ‘Notepad.exe’.

Kamu dapat mengakses notepad dengan beberapa cara sebagai berikut ;

Melalui 'All Programs > Accessories' pada tombol StartMengetik 'notepad' pada kotak dialog search atau pada kotak 'run'Mengetik ‘notepad’ pada jendela 'Command Prompt'

Notepad adalah software editor teks paling simpel yang dapat Anda temukan di versi Windows versi desktop mana pun. Mulai dari Windows versi 1.0 yang rilis pada tahun 1985 sampai dengan Windows 11 (rilis pada tahun 2021) saat ini. Notepad pertama kali rilis pada Agustus 1983 sebagai Multi-Tool Notepad, namun baru diperkenalkan oleh Microsoft pada November 1985 sebagai salah satu paket bundle software Windows versi pertama yaitu Windows Graphic Environment 1.0.

Kelebihan Notepad :

Tampilan simpel/sederhana, menu programmnya mudah dipelajari. Bagi anak yang baru mulai belajar menggunakan komputer, biasanya orang tua atau gurunya akan memperkenalkan Notepad terlebih dahulu, sebelum beranjak ke WordPad atau Microsoft Word. Dalam kurikulum mata pelajaran TIK SD, Notepad sudah dimasukkan ke dalam materi pelajaran kelas 1 SD.Notepad dapat digunakan untuk membuat program-program berskala kecil berbasis DOS atau Command Prompt. Ada kalanya kita memerlukan program-program batch tertentu untuk user, karena kita tidak mungkin menjelaskan proses suatu instruksi yang begitu panjang dan rumit kepada user. Misalnya pada kasus dimana suatu perusahaan memerlukan konfigurasi firewall terhadap sejumlah IP address dan program ‘.exe’ tertentu dengan konfigurasi firewall yang bervariasi di setiap divisi. Sementara di perusahaan tersebut terdapat belasan divisi, yang di dalamnya terdapat puluhan atau ratusan user. Akan lebih cepat bila eksekusinya kita gunakan sebuah program batch yang siap dieksekusi setiap user di perusahaan tersebut, daripada mengkonfigurasinya satu-persatu di setiap komputer.

Kekurangan Notepad :

Tidak seperti Wordpad, Notepad tidak dapat menampilkan dengan baik perpindahan baris teks (line-ending) yang ditulis dalam format UNIX (LF) ataupun Mac (CR). Notepad hanya dapat menampilkan line-ending dalam format Windows (CR+LF) sendiri.Tampilan dan dukungan fitur yang sangat sederhana, menyebabkan para programmer lebih memilih software editor teks Third-Party yang lebih ramai fiturnya, terutama fitur penandaan sintaksnya yang lebih jelas, untuk memudahkan pekerjaan mereka dalam coding. Di antara nama-nama editor teks Third-Party tersebut misalnya Notepad++, PSPad, Geany, Sublime Text, Notepad2, Notepad3, dan yang sedang populer di kalangan programmer Android, Visual Studio Code.

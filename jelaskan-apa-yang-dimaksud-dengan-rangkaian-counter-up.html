12. Multiplexer, Decoder, Flip-Flop, Shift Register, dan Counter merupakan rangkaianbantu jwb ya k​


Jawaban:

Multiplexer :

adalah sebuah komponen elektronika yang berfungsi menghasilkan output dari beberapa pilihan data yang dimasukkan. Multiplexer bisa disebut juga sebagai alat selector. Data input dalam multiplexer (D) bergantung kepada jumlah data selector(S) yang digunakan untuk menghasilkan output.

Decorder :

adalah rangkaian yang berfungsi untuk mengembalikan input menjadi bentuk data dalam format awalnya. Decoder akan menerima banyak input untuk mengubahnya menjadi 1 output, sementara Encoder akan menerima 1 input dan mengubahnyaadalah sirkuit digital terkecil yang berfungsi sebagai rangkaian memori. Flip-flop terdiri menjadi banyak output.

Flip-Flop :

adalah sirkuit digital terkecil yang berfungsi sebagai rangkaian memori. Flip-flop terdiri dari rangkaian gerbang logika dengan 2 kondisi yang stabil, outputnya akan bersifat tetap hingga ada sinyal yang memicu perubahan nilai output tersebut. Terdapat berbagai macam jenis flip-flop, beberapa di antaranya adala SR-FF, JK-FF, D-FF, T-FF, dan JKMS-FF.

Shift Register :

adalah komponen dalam rangkaian digital yang terdiri atas beberapa elemen-elemen memori. Register berfungsi untuk menyimpan nilai data yang dapat dibaca, digeser posisinya, bahkan dihapus. Register dapat dibentuk dari rangkaian sekuensial flip-flop. Register yang terdiri atas n flip-flop mampu menyimpan data sebanyak n bit.

Counter :

adalah salah satu implementasi dari rangkaian flip-flop yang sering digunakan. Semua tipe flip-flop yang dilengkapi saluran Preset, Clock, atau Clear dapat digunakan sebagai komponen rangkaian counter. Contoh aplikasi rangkaian counter adalah jam digital dan stopwatch. Dalam dunia digital, counter dapat dibagi menjadi dua jenis, yaitu synchronous counter dan asynchronous counter.

Pembahasan:

Taktik counter attack atau serangan balik biasanya dilakukan oleh tim-tim yang berada dibawah tekanan, tidak diunggulkan atau minim pemain bintang saat menghadapi tim favorit atau tim bintang. Taktik tersebut cukup populer dan efektif membuat frustrasi tim yang tampil menyerang atau mengandalkan penguasaan bola. Formasi yang digunakan untuk umumnya adalah 4-4-2, 4-4-1-1, 5-3-2, 5-4-1.

Agar dapat memaksimalkan taktik ini, tim biasanya akan menumpuk pemain di zona pertahanan sendiri, hingga di area sebelum kotak penalti. Setelah tim berhasil memancing keluar para pemain lawan untuk maju menyerang, disaat itulah tim segera mengumpan atau membuang bola kedepan untuk disambut oleh penyerang yang gesit plus kemampuan menguasai bola diatas rata-rata untuk melewati para pemain bertahan lawan yang tersisa, bahkan termasuk kiper sendiri untuk segera mencetak gol.

35. Jelaskan perbedaan decoder, encoder dan counter

Encoder

38. Buatkan saya rangkaian proteus counter down e-9-0


Jawaban:

Tentu, berikut adalah contoh rangkaian Proteus untuk membuat sebuah counter countdown dengan tampilan 7 segmen (e-9-0):

Langkah 1: Buka Proteus dan buat proyek baru.

Langkah 2: Tambahkan komponen-komponen berikut ke dalam proyek:

1. Arduino UNO: Pilih dari "Pick Devices" dan cari "Arduino UNO".

2. Breadboard: Pilih dari "Virtual" dan cari "Breadboard".

3. 7-Segment Display (e-9-0): Pilih dari "Pick Devices" dan cari "7-Segment Display (e-9-0)".

4. Resistor 220 Ohm: Pilih dari "Passive" dan cari "Resistor".

5. Push Button: Pilih dari "Virtual" dan cari "Push Button".

Langkah 3: Hubungkan komponen-komponen tersebut seperti pada gambar berikut:

![Counter Down Circuit](https://i.imgur.com/tJWzD7V.png)

Penjelasan:

1. Tambahkan resistor 220 Ohm ke masing-masing pin A, B, C, D, E, F, dan G pada 7-segment display.

2. Sambungkan pin GND pada Arduino ke kaki GND pada Breadboard.

3. Hubungkan pin VCC pada 7-segment display ke 5V pada Arduino.

4. Sambungkan pin D1, D2, D3, D4, D5, D6, dan D7 pada 7-segment display ke pin 2, 3, 4, 5, 6, 7, dan 8 pada Arduino.

5. Tambahkan push button dan hubungkan satu kaki push button ke pin 9 pada Arduino, dan kaki lainnya ke GND pada Breadboard.

Langkah 4: Setelah melakukan koneksi tersebut, Anda dapat memprogram Arduino menggunakan Arduino IDE atau Proteus VSM. Berikut contoh program sederhana menggunakan Arduino IDE:

```arduino

// Library untuk mengontrol 7-segment display

#include <SevSeg.h>

// Konfigurasi pin dan tipe display

SevSeg sevseg;

// Pin untuk push button

const int buttonPin = 9;

// Variabel untuk menyimpan waktu countdown

int count = 9;

void setup() {

// Inisialisasi 7-segment display

sevseg.begin(COMMON_CATHODE, 4, 3, 2, 5, 6, 7, 8, 9, 10, 11, 12, 13);

// Konfigurasi pin push button sebagai input

pinMode(buttonPin, INPUT);

// Tampilkan waktu awal pada 7-segment display

sevseg.setNumber(count);

sevseg.refreshDisplay();

}

void loop() {

// Cek status push button

int buttonState = digitalRead(buttonPin);

// Jika push button ditekan

if (buttonState == HIGH) {

delay(50); // Debouncing delay

// Kurangi waktu countdown

count--;

// Jika waktu countdown mencapai 0, reset ke 9

if (count < 0) {

count = 9;

}

// Update tampilan 7-segment display

sevseg.setNumber(count);

sevseg.refreshDisplay();

}

}

```

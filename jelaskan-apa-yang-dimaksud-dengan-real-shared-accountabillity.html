4. Jelaskan proses menonaktifkan status shared workbook​


Jawaban:

Jika Anda ingin menghilangkan status shared workbook dari file Excel, Anda dapat mengikuti langkah-langkah berikut:

1. Buka file Excel yang ingin Anda nonaktifkan status shared workbook-nya.

2. Klik tab "Review" di ribbon atas, kemudian cari grup "Changes".

3. Pada grup "Changes", cari tombol "Share Workbook", kemudian klik.

4. Akan muncul kotak dialog "Share Workbook". Di dalam kotak dialog ini, pastikan bahwa opsi "Editing" pada tab "Advanced" sedang dipilih.

5. Hilangkan centang pada opsi "Allow changes by more than one user at the same time. This also allows workbook merging." di bawah "Settings".

6. Klik "OK" untuk menonaktifkan status shared workbook pada file Excel.

Setelah itu, file Excel tidak lagi memiliki status shared workbook. Namun, perlu diingat bahwa jika ada pengguna lain yang masih membuka dan mengedit file Excel tersebut dengan status shared workbook, Anda mungkin tidak dapat menghapus status shared workbook sampai pengguna lain menutup file tersebut.

7. Jelaskan apa yang dimaksud dengan isolated I/O dan shared I/O dalam teknik pengalamatan I/O?


Modul I/O adalah suatu komponen dalam sistem komputer yang bertanggung jawab ataspengontrolan sebuah perangkat luar atau lebih dan bertanggung jawab pula dalam pertukaran data antara perangkat luar tersebut dengan memori utama ataupun dengan register – registerCPU.Langkah-langkah Penanganan I/O
1. CPU mengecek staus modulI/O Device2. I/O module mengirimkan  statusnya3. Jika ready, CPU meminta transfer data4. I/O modul mengambil data dari device5. I/O modul transfer data keCPU dalam variasi output yang diinginakan
Metode Pengaksesan I/O
1.MemoriMapped I/ODalam memory-mapped I/O, terdapat ruang tunggal untuk lokasi memori dan perangkat I/O. CPU memperlakukan register status dan register data modul I/O sebagai lokasi memori danmenggunakan instruksi mesin yang sama untuk mengakses baik memori maupun perangkat I/O.Konskuensinya adalah diperlukan saluran tunggal untuk pembacaan dan saluran tunggaluntuk penulisan. Keuntungan memory-mapped I/O adalah efisien dalam pemrograman, namun memakan banyak ruang memori alamat.2. Isolated I/ODalam teknik isolated I/O, dilakukan pemisahan ruang pengalamatan bagi memori dan ruang pengalamatan bagi I/O. Dengan teknik ini diperlukan bus yang dilengkapi dengan saluran pembacaan dan penulisan memori ditambah saluran perintah output. Keuntungan isolated I/Oadalah sedikitnya instruksi I/O.
Metode Operasi sistem I/O
1.    I/O TerprogramPada I/O terprogram, data saling dipertukarkan antara CPU dan modul I/O. CPU mengeksekusi program yang memberikan operasi I/O kepada CPU secara langsung,seperti pemindahan data, pengiriman perintah baca maupun tulis, dan monitoring perangkat. Kelemahan teknik ini adalah CPU akan menunggu sampai operasi I/O selesai dilakukan modul I/O sehingga akan membuang waktu, apalagi CPU lebih cepat proses operasinya.2. I/O Instruksi(Demand Driven)driven I/O memungkinkan proses tidak membuang – buang waktu. Prosesnya adalah CPU mengeluarkan perintah I/O pada modul I/O, bersamaan perintah I/O dijalankan modul I/O maka CPU akan melakukan eksekusi perintah – perintah lainnya.Apabila modul I/O telah selesai menjalankan instruksi yang diberikan padanya akan melakukan interupsi pada CPU bahwa tugasnya telah selesai.
3. Direct Memory Access (DMA)Teknik yang dijelaskan sebelumnya yaitu I/O terprogram dan Interrupt-Driven I/O memiliki kelemahan, yaitu proses yang terjadi pada modul I/O masih melibatkan CPU secara langsung. Hal ini berimplikasi pada :• Kelajuan transfer I/O yang tergantung pada kecepatan operasi CPU.• Kerja CPU terganggu karena adanya interupsi secara langsung.Bertolak dari kelemahan di atas, apalagi untuk menangani transfer data bervolume besar dikembangkan teknik yang lebih baik, dikenal dengan Direct Memory Access(DMA).Prinsip kerja DMA adalah CPU akan mendelegasikan kerja I/O kepada DMA, CPU hanya akan terlibat pada awal proses untuk memberikan instruksi lengkap pada DMA dan Akhir proses saja. Dengan demikian CPU dapat menjalankan proses lainnya tanpa banyak terganggu dengan interupsi.

12. apa yang dimaksud dengan bilangan real rasional dan bilangan real irrasional


Bilangan real rasional adalah bilangan yang dapat dinyatakan dalam a/b, dan a,b adalah rasional.

Misalnya:
1.) 3 merupakan real rasional karena 3 = 3/1
2.) 1/6 merupakan real rasional (jelas)
3.) 6,47 merupakan real rasional karena 6,47 = 647/100

Bilangan real irasional adalah bilangan real yang tidak dapat dinyatakan dalam a/b
Misalnya pada akar kuadrat:
1.) √3 tidak dapat dinyatakan dalam a/b
Pada konstanta irasional lainnya
2.) π tidak dapat dinyatakan dalam a/b
3.) ∛9 tidak dapat dinyatakan dalam a/b
bilangan rasional adalah bilangan yang dapat dinyatakan menjadi a/b
contohnya : 1/2 = 0,5
bilangan irasional adalah bilangan yang tidak dapat dinyatakan menjadi a/b
contohnya : √2 = 1,4........
√2 tidak dapat dikatakan rasional karena √2 tidak dapat diubah ke bentuk bilangan pasti

32. Apa yang dimaksud dengan shared services organization


Shared Services Organization (SSO) adalah sebuah konsep dalam manajemen korporat di mana beberapa departemen atau unit bisnis berbeda dalam suatu perusahaan bekerja sama dengan membagi sumber daya dan layanan tertentu, seperti keuangan, SDM, TI, dan lainnya, untuk mengoptimalkan efisiensi dan kinerja perusahaan secara keseluruhan. SSO dapat membantu perusahaan untuk menghemat biaya operasional, meningkatkan kualitas layanan, dan memberikan visibilitas yang lebih baik dalam pengelolaan bisnis. Organisasi berbagi ini sering kali memiliki sistem yang terpusat untuk pengelolaan data dan informasi, serta menerapkan standar proses dan metrik untuk menilai kinerja mereka.

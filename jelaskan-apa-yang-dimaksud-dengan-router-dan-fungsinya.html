1. salah satu fungsi router sebagai chace memory, jelaskan maksudnya


cache memory itu sejenis history dalam hal ini dari client. Jadi pada saat client ingin membuka hal yang sama, maka server dapat mnyediakannya dengan cepat.

Contoh :
ali ingin mengakses web google.com, otomatis permintaan itu melalui server, nah si server ini jika dia baru mengetahui ada web yang bernama google.com maka server akan menyimpan google.com di file cache milik server. Sehingga saat ali atau siapapun ingin mengakses google.com, mereka tidak perlu mengirim request ke google.com, server akan langsung menyediakannya, dan pastinya lebih cepat.

27. apa fungsi dari ROUTER?


Router, seperti namanya, adalah 'penemu rute/jalur' pada jaringan. Router menghubungkan dua jaringan dengan cara menemukan jalur yang paling baik yang akan dilalui paket data berdasarkan alamat IP yang ada dalam data, baik alamat IP asal maupun tujuan. Rute 'yang paling baik' yang dimaksud adalah rute yang tidak sibuk, router akan memilih jalur yang bersih/tidak sibuk.

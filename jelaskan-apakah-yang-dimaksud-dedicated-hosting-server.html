7. Seberapa tahukan anda tentang dedicated server dan berikan pemahaman kalian tentang kenapa sebuah perusahaan cenderung memilih dedicated server dibanding 2 hosting lainya?


Dedicated server merupakan penyewaan satu server secara utuh tanpa kita harus membagi dengan user yang lain, sehingga hanya perusahaan itu sendiri yang menempati dan menggunakan dedicated server tersebut. Perusaahaan berkuasa penuh atas pengelolaan dedicated server tersebut termasuk pemilihan sistem operasi, hardware, dan sebagainya. Dedicated server mempunyai kelebihan sebagai berikut

1. Akses Penuh (Tidak di Bagi)

Ketika anda menggunakan dedicated server maka hak akses sepenuhnya milik anda. Tak terbagi dengan user lain. Hal ini membuat anda memiliki sumber daya penuh dari server tunggal. Selain itu, anda juga bisa dengan bebas dalam hal konfigurasi serta penggunaannya.

2. Keamanan

Menggunakan dedicated server sangat baik untuk website yang memiliki jumlah pengunjung cukup banyak, karena ruang yang tersedia tidak terbagi dengan website lain yang artinya hal ini akan menjamin keamanan dari website anda sendiri.  

3. Alamat IP Sendiri

Berbeda dengan Share hosting, memiliki dedicated server berarti anda bisa memiliki alamat IP unik untuk anda sendiri. Berbeda dengan shared hosting yang berbagi alamat ip dengan pengguna yang lain.

4. Bisa Cek Via Remote

Memiliki dedicated server bukan berarti anda harus mengeceknya secara langsung, justru anda bisa mengontrolnya dengan cara mudah yaitu dengan cara remote jarak jauh. Anda tidak perlu datang dan mengecek secara langsung pada fisik server, itu merepotkan yah. Cukup awasi saja secara remote.

Shared Hosting

Shared hosting adalah sebuah layanan web hosting dimana sebuah akun hosting akan ditaruh bersama dengan beberapa akun hosting lain didalam satu server yang sama. Sehingga dengan kata lain, shared hosting adalah sebuah server yang digunakan oleh banyak user atau akun hosting, menggunakan resource server yang sama baik itu prossesor, ram maupun storage. Tentu saja dengan banyaknya akun hosting yang menggunakan resource sama, maka website yang anda miliki tidak akan selalu bisa berjalan dengan stabil, tidak jarang website yang anda miliki akan sering down karena disebabkan user atau akun hosting lain sedang mendapat traffic pengunjung yang tinggi. Oleh karena itu, shared hosting tidak cocok untuk digunakan bagi anda yang memiliki usaha besar.

Cloud Hosting

Cloud Hosting dengan Shared Hosting adalah sebuah layanan web hosting namun terdapat perbedaan yang membuat teknologi cloud hosting jauh lebih unggul serta lebih sering digunakan oleh penyedia layanan hosting terbaik dibanding dengan shared hosting yang sudah mulai ditinggalkan.

Baca juga: Tips Cara Memilih Web Hosting Terbaik Untuk Website Anda

Hal ini karena teknologi cloud dianggap jauh lebih efesien ketimbang dengan shared hosting. Cloud hosting adalah sebuah layanan web hosting yang memiliki teknologi dimana sebuah akun hosting dapat terhubung pada banyak server, sehingga pengguna akun hosting tersebut tidak perlu takut dengan performa hostingnya down karena ketika hosting anda mendapatkan trafik pengguna yang tinggi, maka hosting akan otomatis berpindah ke server yang tidak memiliki trafik tinggi, alhasil hosting anda pun tidak akan bermasalah karena banyaknya trafik yang masuk ke website anda.

Virtual Private Server (VPS)

VPS adalah Virtual Private server. VPS dapat dikatakan sebagai titik tengah antara shared hosting dengan dedicated server. VPS adalah sebuah server yang menggunakan teknologi virtual untuk membagi hardware fisik menjadi beberapa virtual server.

Baca juga: Colocation Server Murah atau Dedicated Server Murah

Setiap VPS dapat menjalankan sistem operasinya masing-masing dan juga dapat di restart atau reboot terpisah meski didalam satu server terdapat beberapa virtual machine didalam server. Anda mendapatkan akses penuh terhadap VPS yang anda miliki, namun tetap saja anda harus berbagi resource dengan user lain, apalagi ketika user lain menggunakan VPS dengan sangat boros, maka anda akan ikut terkena imbasnya.

Dedicated Server

Sementara itu, dedicated server adalah sebuah layanan penyewaaan sebuah server kepada seorang user atau pengguna. Sehingga hanya anda sendiri yang akan menggunakan dan menempati sebuah server tersebut. Anda memiliki hak yang jauh lebih tinggi jika dibandingkan dengan VPS. Dedicated Server sangat cocok digunakan oleh anda yang memiliki

17. Keunggulan dan kelemahan Dedicated Hosting Server yaitu :


Kelebihan:
1. Akses Penuh 
Dedicated server memeberikan kebebasan kepada penyewa untuk memiliki hak akses yang dimiliki 100%. Menariknya lagi adalah Dedicated Server juga akan memudahkan dan membebaskan penggunanya untuk melakukan konfigurasi berdasarkan bentuk spesifikasi yang dibutuhkan.

2. Aman 
Kelebihan dari penggunaan Dedicated Server selanjutnya adalah apabila terjadi kerusakan pada server, Anda tidak bertanggung jawab pada kerusakan tersebut. Sehingga sangat aman bagi Anda untuk mengoperasikan Dedicated Server tersebut.

3. Efektif, Bisa Menggunakan Remote 
Kelebihan lainnya adalah penyewa bisa mengontrol server menggunakan remote saja. Anda tidak perlu melakukan pengawasan pada fisik server secara langsung. Sehingga tidak akan merepotkan atau membebani Anda selama mengelola website.

4. Tidak Terganggu Aktifitas Lain 
Ketika Anda menggunakan Dedicated Server, Anda tidak perlu merasa terganggu dengan klien yang lain. Mengapa seperti itu? Sebab hosting dengan jenis Dedicated Server hanya digunakan oleh 1 orang pengguna saja. Jadi, tidak akan mengganggu kinerja website apabila sewaktu-waktu jumlah pengunjungnya meningkat. Hosting ini juga memungkinkan penggunanya untuk mengelola banyak domain pada satu server yang dimiliki. Dimana manajemen website juga akan semakin maksimal karena Anda lah yang mempunyai kontrol dan kuasa penuh terhadap server tersebut.

Kekurangan:
1. Harganya Mahal 
Kekurangan yang dimiliki oleh hosting dengan tipe Dedicated Server adalah biaya yang dibutuhkan melakukan masuk ke dalam kategori mahal. Semakin tinggi spesifikasi server yang diinginkan maka semakin mahal pula harga yang harus dibayarkan. Berbeda dengan share hosting yang memungkinkan Anda untuk membayar sewa bersama dengan pengguna lainnya.

2. Skill yang Advanced 
Kekurangan lain dari penggunakan Dedicated Server adalah penyewa harus memahami kinerja dan cara kontrol server. Hal tersebut sangat penting agar Anda bisa mengelola dedicated server dengan maksimal. Apalagi, ada cukup banyak pengaturan di berbagai aplikasi yang harus dilakukan. Sehingga Anda harus benar-benar mempelajari pengelolaan data menggunakan dedicated server sebelum menyewanya.

Bagaimana Cara Kerja Web Hosting?

Ketika Anda mengakses sebuah website (baik menggunakan IP Address maupun nama domain) melalui browser yang menggunakan network protocol HTTP, internet akan mengirimkan permintaan akses kepada server hosting. Web server adalah program software yang bertanggung jawab untuk menangani permintaan dari internet. Saat mengetik URL ke bilah alamat browser Anda, itu akan diterjemahkan ke dalam permintaan yang diarahkan ke komputer yang menghosting situs web yang Anda cari. Intinya, Web server akan membaca permintaan, mencari tahu apa aplikasi lain yang perlu dijalankan atau file yang diakses, dan kemudian setelah selesai, ia mengirim tanggapan kembali ke browser. Tanggapan yang dikirim kembali adalah (biasanya) halaman situs web yang ingin Anda lihat. Software web server bertindak sebagai mediator antara internet dan file-file di server.

USER/pengguna yang akan mengakses suatu website berupa URL melalui WEB BROWSER (yaitu media untuk menuju URL yang diakses), kemudian WEB BROWSERtersebut mengirimkan permintaan/ request berupa HTTP REQUEST kepada WEB SERVER melalui layer-layer TCP/IP,kemudian WEB SERVER memberi WEB FILES yang direquest jika ada.

WEB FILES yang telah diberikan tadi tidak langsung ditampilkan/di-display begitu saja, namun WEB SERVERmemberikan respon kembali ke WEB BROWSER melaluiHTTP RESPONE yang juga melalui layer-layer TCP/IP, yang kemudian baru di terima oleh WEB BROWSER, dan kemudian dikirmkan kepada USER berupa DISPLAY.

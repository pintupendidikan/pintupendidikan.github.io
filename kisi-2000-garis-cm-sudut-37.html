1. Seberkas cahaya jatuh tegak lurus pada kisi yang terdiri dari 2000 garis tiap cm. sudut deviasi orde ke 3 adalah 37. panjang gelombang cahaya yang digunakan adalah .....


Panjang gelombang cahaya yang digunakan adalah 10⁻⁶ meter. Panjang gelombang tersebut dapat ditentukan dengan menerapkan konsep kisi difraksi.

Penjelasan dengan langkah-langkah:

Diketahui : N = 2000 garis/cm ≈ 2 × 10⁵ garis/m

                   m = 3

                   ∅ = 37°

Ditanya : λ ?

Jawab :

Langkah 1 : menentukan rumus untuk menyelesaikan soal  

        Soal di atas merupakan penerapan materi kisi difraksi. Persamaan yang digunakan adalah sebagai berikut.

        d × sin ∅ = m × λ

        [tex]\frac{1}{N}[/tex] × sin ∅ = m × λ

        Keterangan : d = lebar celah

                               N = banyaknya kisi dalam satuan panjang

                               ∅ = sudut bias

                               m = orde difraksi

                               λ = panjang gelombang

Langkah 2 : menentukan panjang gelombang cahaya

       [tex]\frac{1}{N}[/tex] × sin ∅ = m × λ

       [tex]\frac{1}{2x10^{5} }[/tex] × sin 37° = 3 × λ

        λ =  [tex]\frac{1}{2x10^{5}x3 }[/tex] × sin 37°

        λ =[tex]\frac{1}{6x10^{5} }[/tex] x 0,6

        λ = 10⁻⁶ meter

- sudut difraksi orde dua --> θ = 37

9. seberkas cahaya dengan panjang gelombang 1500nm didifraksikan oleh kisi dengan 2000 garis/cm. tentukan orde maksimum yang mungkin terlihat pada layar​


Cahaya adalah gelombang elektromagnetik yang dapat merambat tanpa memerlukan media perantara. Sifat-sifat cahaya adalah dapat dimantulkan, dibiaskan, diinterferensikan dan dibelokkan (difraksi). Difraksi adalah pembelokan arah cahaya karena melewati celah-celah sempit sehingga terbentuklah pola terang dan gelap. Ada dua peristiwa difraksi yang bisa terjadi yakni difraksi pada celah tunggal dan difraksi pada celah kisi.

difraksi celah tunggal

pola terang akan terbentuk pada layar jika

[tex]\frac{p x d}{l}[/tex] = (2n+1) x 1/2λ

pola gelap akan terbentuk jika memenuhi persamaan

[tex]\frac{p x d}{l}[/tex] = n x λ

difraksi celah kisi

pola terang akan terbentuk jika memenuhi persamaan

[tex]\frac{p x d}{l}[/tex] = n x λ

atau

d sin θ = n x λ

dengan  d = 1/N

pola gelap akan terbentuk jika memenuhi persamaan

[tex]\frac{p x d}{l}[/tex] = (2n +1) x λ

keterangan

p = jarak antar pola terhadap terang pusat

d = lebar celah kisi

l = jarak layar ke kisi

n = orde difraksi

λ = panjang gelombang

Pembahasan

Berdasarkan pembahasan diatas maka

diketahui

λ = 1500 nm = 1500 x [tex]10^{-9}[/tex]

N = 2000 garis/cm

ditanyak

n ....?

penyelesaian

d sin θ = n λ

hitung besarnya d dengan menggunakan persamaan d = 1/N, sehingga

d = 1/2000

d = 0,0005 cm

d = 5 x [tex]10^{-4}[/tex] cm

d = 5 x [tex]10^{-6}[/tex] m

substitusikan nilai d ke persamaan

d sin θ = n λ

dan anggap θ= 90, sehingga

5 x [tex]10^{-6}[/tex] m x sin 90 = n x 1500 x [tex]10^{-9}[/tex]

5 x [tex]10^{-6}[/tex] m x 1 = n x 1500 x [tex]10^{-9}[/tex]

n = 5 x [tex]10^{-6}[/tex] : 1500 x [tex]10^{-9}[/tex]

n = 0,003 x [tex]10^{3}[/tex]

n = 3

10. cahaya hijau (panjang gelombang = 5400 angstrom) didifraksikan oleh kisi dengan 2000 garis/cm. spektra orde tertinggi yang diamati adalah...


λ = 5400 angstrom = 5400 x 10⁻¹⁰ m = 5,4 x 10⁻⁷ m
d = 1 / N
   = 1 / (2 x 10³ garis/cm) = 5 x 10⁻⁴ cm = 5 x 10⁻⁶ m

d sin θn = nλ
sin θn = nλ / d
           = (5,4 x 10⁻⁷ m)n / (5 x 10⁻⁶ m)
           = (1,08 x 10⁻¹) n
Nilai maksimum fungsi sin adalah 1
1 = (1,08 x 10⁻¹) n
n = 1 / (1,08 x 10⁻¹)
   ≈ 9

20. seberkas cahaya jatuh tegak lurus pada kisi yang terdiri atas 2000 garis tiap cm.sudut bias ke-37°.panjang gelombang cahaya yang digunakan adalah


Difraksi cahaya merupakan pelenturan cahaya akibat cahaya melalui celah kecil. Pada kisi difraksi, cahaya melalui celah – celah kecil yang jumlahnya banyak sehingga menghasilkan pola gelap terang yang ditangkap oleh layar.

 

Dari soal di atas dapat diketahui bahwa:

- Terjadi difraksi kisi/ celah majemuk

- sudut difraksi orde dua --> θ = 37

- konstanta kisi --> N = 2000 garis/ cm = 200.000 garis/ m

 

Sedangkan yang menjadi permasalahan pada soal adalah panjang gelombang.

 

Mula – mula kita cari n untuk terang orde dua yaitu n = 2.

 

Kemudian kita cari dahulu d:

d = 1/N = 1/500.000 = 2. 10^{-6}

 

Untuk mencari panjang gelombang digunakan rumus berikut:

nλ = d sin θ

2. λ = 5. 10^{-6}. Sin 37

λ = 1,5. 10^{-6} m = 1500. 10^{-9} nm = 1500 nm

 

Berdasarkan uraian di atas, didapat bahwa panjang gelombang yang digunakan 1500 nm. 

 

 

PEMBAHASAN LEBIH LANJUT :

 

Persamaan umum yang digunakan pada interferensi dan difraksi cahaya yaitu:

 

nλ = d.p/l

 

 

 

atau

 

nλ = d sin θ

 

keterangan:

n = orde terang/ gelap

λ = panjang gelombang

d = jarak celah

p = jarak antar pola

l = jarak layar ke celah

θ = sudut deviasi

 

Ketentuan n pada kisi difraksi:

Pola terang: n = 3/2, 5/2, 7/2, 9/2, dst.

Pola gelap: n = 1, 2, 3, 4, dst..

 

Pada kisi difraksi terdapat jumlah celah/ goresan tiap meter yang disebut N

N = 1/d

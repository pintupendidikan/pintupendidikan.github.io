[(—3/2x) pangkat –3]²

= ((-3/2x)^-3)²

= ((-3)^-3 /(2x)^-3)²

= (-3)^(-3x2) / (2^(-3x2) x^(-3x2))

= (-3)^-6 / (2^(-6) x^(-6))

= 2^6 x^6 / (-3)^6

= 64x^6 / 729

jadi, hasil akhirnya adalah 64 x pangkat 6 per 729

= Angka satuan pada bilangan 3²⁰⁰¹

Setelah mencari angka satuan pada bilangan 3ⁿ, ini polanya.

3, 9, 7, 1, 3, 9, 7, 1, ....

Misalkan U₀ = 1

U₁ = 3

U₂ = 9

U₃ = 7

Lihat, U₄ = U₀ dan 4 dibagi 4 bersisa 0.

2001 dibagi 4 bersisa 1 jadi U₂₀₀₁ = U₁ = 3

Jadi, angka satuan pada bilangan 2013²⁰⁰¹ adalah 3

30. tentukan hasil bilangan berpangkat tersebut:(2/5)³ = tolong bantu jawab, soalnya mau dikumpulkan ​

Untuk menghitung hasil bilangan berpangkat, kita akan memangkatkan angka dan memangkatkan penjumlahan mereka. Jadi, (2/5)^3 dapat dihitung sebagai berikut:

(2/5)^3 = (2/5) x (2/5) x (2/5)

= (2 x 2 x 2) / (5 x 5 x 5)

= 8 / 125

Jadi, hasil dari (2/5)^3 adalah 8/125.

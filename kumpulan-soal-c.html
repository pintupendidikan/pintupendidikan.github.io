12. C.
2A  - {3A - (B - 2C)} - C
= 2(3x + 1) - {3(3x + 1) - ((-2x + 3) - 2(4x - 5)} - (4x - 5)
= 6x + 2 - {9x + 3 - (-2x + 3 - 8x + 10)} - 4x + 5
= 6x + 2 - {9x + 3 - (-10x + 13)} - 4x + 5
= 6x + 2 - (19x - 10) - 4x + 5
= -17x + 17

Kelas : VII SMP
Bab : Aljabar dasar

2A - (3A - (B - 2C)) - C
= 2(3x + 1) - [3(3x + 1) - (-2x + 3 - 8x + 10)] - (4x - 5)
= 6x + 2 - [9x + 3 - (-10x + 13)] - 4x + 5
= 6x + 2 - (19x - 10) - 4x + 5
= -17x + 17

9. PNS yang kreatif adalah PNS yang


Jawaban:

Penjelasan:

PNS yang kreatif adalah PNS yang memiliki dedikasi tinggi terhadap pekerjaannya dan mampu mengatur waktu untuk kegiatan selain pekerjaan.

PNS yang kreatif setidaknya memiliki sifat jujur dan integritas tinggi serta mampu menyesuaikan jam kerja dan jam istirahat serta mampu menjaga privasi pemerintahan.

PNS hendaknya siap menerima kritik maupun saran dan siap untuk ditempatkan di berbagai posisi sesuai dengan jabatan atau golongan yang dimiliki.

Pembahasan

PNS memiliki tanggung jawab yang berat dan tugas yang banyak. Wajar jika seorang PNS harus kreatif dan aktif di lingkungan kerjanya. Jika PNS tidak mampu beradaptasi dengan lingkungan kerjanya maka terjadi penurunan kinerja yang menyebabkan banyak terjadi permasalahan di institusi pemerintahan atau tempat PNS itu bekerja.

PNS yang kreatif harus mampu mengkoordinasikan tugas yang didapat agar tidak mengganggu waktu lainnya dan mampu memberikan pelayanan maksimal untuk pemerintah guna menghasilkan birokrasi yang baik dan bermartabat. PNS juga bisa memiliki pekerjaan sampingan tetapi diluar jam kerja sebagai PNS untuk menghindari terjadinya bentrok jika jam kerja PNS digunakan untuk pekerjaan lain. Jika hal ini terjadi maka akan menimbulkan banyak masalah khususnya kinerja PNS itu sendiri.

14. urusan pemerintah pusat dan daerah dalam uu no.23 th.2014 adalah(?).Tolong jawab segera,soalnya dikumpulkan besuk!"makasih"


Berdasarkan UU Nomor 23 tahun 2014 klasifikasi urusan pemerintahan terdiri dari 3 urusan yakni urusan pemerintahan absolut, urusan pemerintahan konkuren, dan urusan pemerintahan umum. Urusan pemerintahan absolut adalah Urusan Pemerintahan yang sepenuhnya menjadi kewenangan Pemerintah Pusat. Urusan pemerintahan konkuren adalah Urusan Pemerintahan yang dibagi antara Pemerintah Pusat dan Daerah provinsi dan Daerah kabupaten/kota. Urusan pemerintahan umum adalah Urusan Pemerintahan yang menjadi kewenangan Presiden sebagai kepala pemerintahan.

21. Tentukan a. banyak angka 0 pada hasil bagi 2014 2014 2014 dibagi 2014 b. apabila angka 2 1 0 dan 4 masing-masing terdapat 300 angka seperti pola soal a berapakah hasil baginya ketika dibagi 2014


a. Banyak angka 0 pada hasil bagi 2014 2014 2014 : 2014 adalah 6.

b.  Angka 2, 1, 0, dan 4 masing-masing terdapat 300 angka seperti pola soal a, maka hasil baginya ketika dibagi 2014 adalah 897.  

Penjelasan dengan langkah-langkah

Diketahui :

Bilangan 2014 2014 2014

Ditanya :

a. Banyak angka 0 hasil dari 2014 2014 2014 : 2014.

b. Angka 2, 1, 0, dan 4 masing-masing terdapat 300 angka seperti pola soal a, maka hasil baginya ketika dibagi 2014.

Jawab :

a. Menentukan banyak angka 0

Hasil bagi dari 2014 2014 2014 : 2014 = 100010001

Banyak angka 0 pada 100010001 adalah 6.

b. Menentukan banyak angka 0 pada susunan pengulangan 300 kali

Berdasarkan pola soal a bisa kita buat.

2014 2014 2014 : 2014 = 100010001 ⇒ 3 susunan, nolnya 6

2014 2014 2014 2014 : 2014 = 1000100010001 ⇒ 4 susunan, nolnya 9

2014 2014 2014 2014 : 2014  = 10001000100010001 ⇒ 5 susunan, nolnya 12.

Karena selisih banyak 0 tiap susunan ada 3 dan n = banyak susunan penulangan, maka dapat kita buat rumusnya 3 × n - 3.

Banyak 0 pada susunan 300

= 3n - 3

= 3 × 300 - 3

= 900 - 3

= 897

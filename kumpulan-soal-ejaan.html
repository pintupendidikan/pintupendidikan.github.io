Pembahasan

Kata atau nama Anthony terdiri dari tujuh (7) huruf, di antaranya adalah:

A = eiN = enT = tiH = eijO = ouN = enY = wai

19. tolong bantu saya mengejakan soal nomer 2 & 3 ......


Diketahui:
I₀ = 200 cm = 0,2 m
T₀= 40° C
l₁ = 200,1 cm = 0,2001 m
α = 10⁻⁵/°C

Ditanyakan: T₂

Jawab:
         l₁ = l₀ ( 1+ α.ΔT)
0,2001 = 0,200 ( 1 + 10⁻⁵ . ΔT)
1,0005 = 1 + 10⁻⁵. ΔT
0,0005 = 10⁻⁵.ΔT
       ΔT = 50
 T₁ - T₀ = 50
 T₁ - 40 = 50
        T₁ = 90°C



Diketahui:
I₀ = 20 m
T₀= 25° C
α = 1,2 . 10⁻⁵/°C
T₂ = 55
ΔT = 55-25 = 30°C

Ditanyakan: Δl

Jawab:
l₁ = l₀ ( 1+ α.ΔT)
l₁ = 20 ( 1 + 1,2 .10⁻⁵ . 30)
   = 20 ( 1 + 3,6 .10⁻⁴)
   = 20 . 1,00036
   = 20,0072 m

Δl = l₁ - l₀
    = 20,0072 - 20
    = 0,0072 m

22. bagaimana cara mengejakan soal operasi hitung pada aljabar ?


pada penjumlahan
*komutatif= a+b=b+a
*asosiatif=a+(b+c)=(a+b)+c
*distributif=ab+ac=a(b+c)=(b+c)a

operasi penjumlahan bentuk aljabar dapat dilaksanakan jika dan hanya jika SUKU SUKUNYA SEJENIS
Agar bisa menjumlahkan dan mengurangkan dalam bentuk aljabar, variabel dan pangkatnya itu harus sama

#Penjumlahan aljabar :
1. 2a + a = 2a + 1a = 3a
2. -b + 3b = 2b
3. a + c ≠ (tidak bisa dijumlahkan karena variabel tidak sama)
4. p² + 2p² = 3p²
5. p² + p³ ≠ (tidak bisa dijumlahkan karena pangkat tidak sama)
6. x² + 2x - 3 dan -5x² - 4x + 2
[kata "dan" diubah menjadi +]
= x² + 2x - 3 + -5x² - 4x + 2
[urutkan dengan yang variabel dan pangkatnya sama]
= x² - 5x² + 2x - 4x - 3 + 2 
= -4x² - 2x + 1
7. 2p - q - 5 dan p + 7q - 10
= 2p - q - 5 + p + 7q - 10
= 2p + p - q + 7q - 5 + 10
= 3p + 6q + 5

#Pengurangan aljabar :
1. 5a - (-3a) = 5a + 3a = 8a
2. 3b - 7b = -4b
3. 2c - a ≠ (tidak bisa dikurang karena jumlah tidak sama)
4. -3p² + p² = -2p²
5. p³ - p ≠ (tidak bisa dikurang karena pangkat tidak sama
6. x² + 2x - 3 dan -5x² - 4x + 2
[kata "dan" diubah menjadi -]
= x² + 2x - 3 - (-5x² - 4x + 2)
= x² + 2x - 3 + 5x² + 4x - 2
= x² + 5x² + 2x + 4x - 3 - 2
= 6x² + 6x - 5
7. 2xy + 4x - 3y dan -3xy + x - 2y
= 2xy + 4x - 3y - (-3xy + x - 2y)
= 2xy + 4x - 3y + 3xy - x + 2y   
= 2xy + 3xy + 4x - x - 3y + 2y
= 5xy + 3x - y
8. -3xy + x - 2y dari 2xy + 4x - 3y
[kata "dari" diubah menjadi - dan dibalekkan]
= 2xy + 4x - 3y - (-3xy + x - 2y)
= 2xy + 4x - 3y + 3xy - x + 2y
[urutkan dengan yang variabel dan pangkatnya sama]
=  2xy + 3xy + 4x - x - 3y + 2x
9. y² - 5x + 4 dari 2y² - 3y + 7
= 2y² - 3y + 7 - (y² - 5y + 4)
= 2y² - 3y + 7 - y² + 5y - 4
= 2y² - y² - 3y + 5y + 7 - 4
= y² + 2y + 3

Penjelasan:

Kata baku adalah kata yang digunakan sesuai aturan atau kaidah berbahasa Indonesia yang sudah ditentukan sebelumnya.

Selain itu, penggunaan kata baku dapat dilihat dari penggunaanya yang sudah sesuai EYD (Ejaan yang Disempurnakan). Penggunaaan kata baku ini biasanya digunakan untuk pengungkapan bahasa yang bersifat resmi, dalam bentuk surat resmi maupun naskah pidato.

Ciri-ciri Kata baku

1. Tidak dipengaruhi bahasa daerah tertentu.

2. Tidak dipengaruhi bahasa asing.

3. Bukan bahasa percakapan.

4. Pemakaian imbuhan pada kata bersifat eksplisit.

5. Pemakaian kata sesuai dengan konteks kalimat6. Kata baku bukan kata rancu

7. Kata baku tidak mengandung hiperkorek.

8. Tidak mengandung pleonase.

Kata tidak baku merupakan kebalikan dari kata baku, yang penggunaanya tidak sesuai aturan dan kaidah berbahasa Indonesia yang sudah ditentukan sebelumnya.

Ciri-ciri Kata Tidak Baku

1. Umumnya digunakan dalam bahasa sehari-hari.

2. Dipengaruhi bahasa daerah dan bahasa asing tertentu.

3. Dipengaruhi dengan perkembangan zaman.

4. Bentuknya dapat berubah-ubah.

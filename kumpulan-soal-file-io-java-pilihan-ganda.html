34. cara menggandakan file


Berikut ini adalah beberapa cara menggandakan file pada Windows Explorer:

Cara 1

Pilih file yang ingin digandakan.Pilih menu edit, lalu pilih copy.Masuk ke folder atau direktori tujuan.Pilih menu edit, lalu pilih paste.

Cara 2

Klik kanan pada file yang ingin digandakan.Pilih copy.Masuk ke folder atau direktori tujuan.Pilih paste.

Cara 3

Klik file yang ingin digandakan.Tekan tombol Ctrl dan C secara bersamaan.Masuk ke folder atau direktori tujuan.Tekan tombol Ctrl dan V secara bersamaan.

Pembahasan

Manajemen file merupakan suatu sistem yang berfungsi untuk mengatur file-file ke dalam folder dalam sebuah komputer. Sistem tersebut juga dapat digunakan untuk menemukan, memindahkan, menyalin, dan tentu saja membuka file. File yang dimaksud sendiri dapat berupa file gambar, audio, video, teks. program, dan sebagainya.

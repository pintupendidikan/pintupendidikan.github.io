5. 10 contoh percakapan if clause reminder, an imperative, suggestion dan general truth dan dream


IF CLAUSE AS REMINDER, IMPERATIVE, SUGGESTION AND GENERAL TRUTH

 

Jawaban  Pendahuluan  

If clause atau Conditional sentence selain digunakan sebagai sebuah perumpamaan juga bisa digunakan sebagai pengingat (reminder), saran (suggestion),  perintah (imperative), impian (dream) dan fakta (general truth).

Pembahasan  

If Clause sebagai Reminder atau Pengingat

If clause sebagai reminder digunakan untuk mengingatkan seseorang terhadap sesuatu dengan tujuan agar mendapatkan hasil yang terbaik.

Rumus dalam If clause as reminder adalah:

If + Simple Present, Subject + have/has to atau must + V1

Contoh:

1. If you want to get the highest score, you must study harder.

2. If you want to be a champion, you have to train hard.

3. If she wants to visit me, she must come to my house around 4 pm.


If Clause sebagai Imperative atau Perintah

If clause yang digunakan sebagai perintah bertujuan agar seseorang mematuhi suatu peraturan atau perintah. Kalau orang tersebut mengikuti perintahnya, maka akan hasil yang baik untuknya.

Rumus untuk If clause as imperative:

If + Simple present, V1 + Object / Adverb (imperative)

Contoh:

1. If you want to pass the University enrollment test, stop playing with your mobile phone!

2. If you want to be healthy, eat some nutritious foods and fruits!

3. If you want to buy a car, start saving your money!


If Clause sebagai Suggestion atau Saran

If Clause sebagai suggestion digunakan untuk memberi saran agar seseorang mendapatkan apa yang dia inginkan.

Rumus untuk If clause as Suggestion:

If + Simple Present, S + should/should not + V1

Contoh:

1. If you want to get the highest score, you should study harder.

2. If you want to speak English fluently, you should do more conversation with native English.

3. If you want to be healthy. you should eat more fruits.


If Clause sebagai General Truth atau Fakta

If clause sebagai general truth sering disebut juga sebagai conditional tipe zero. Ungkapan ini menyatakan sebuah fakta. Biasanya hubungan sebab-akibat berlaku dalam tipe ini.

Rumus If clause as General Truth:

If + simple present, simple present

Contoh:

1. If you play with fire, you get burned.

2. If you heat the ice, it melts

3. If you mix yellow and blue, you get green.


If Clause sebagai Impian (Dream)

If clause sebagai dream atau impian digunakan untuk menyatakan sebuah impian dari seseorang di masa depan. If clause ini sering disebut conditional tipe satu.

Rumus If clause as Dream:

If simple present, simple future

Contohnya:

1. If I have much money, I will travel around the world.

2. If I marry that girl, I will be very happy.

3. If you can't come, They will be sad

Pembahasan Conditional Sentences Type 1, 2, 3, and 0

1. First Conditional

First Conditional digunakan untuk pengandaian ke masa depan yang belum dan mungkin terjadi.

POLA kalimat: If + Present Tense, will + infinitive

Contoh:  

If you visit your grandma, she will give you a delicious cake. If my mother knows this, she will be angry.

2. Second Conditional

Second Conditional digunakan untuk pengandaian ke masa lalu yang tidak mungkin terjadi di masa sekarang. Dengan kata lain, sesuatu dapat terjadi saat ini jika sesuatu yang lain di masa lalu dilakukan.

POLA kalimat: If + Past Tense, would/could + infinitive

Contoh:

If we took U turn, we would not get lost. If I were a president, I would give high salary for teachers.

3. Third Conditional

Third Conditional digunakan untuk pengandaian ke masa lalu yang tidak mungkin karena sudah tidak terjadi (di masa lalu).

POLA kalimat: If + Past Perfect Tense, would/could have + past participle

Contoh:

If you had warned me, I wouldn't have gotten an accident. If she had come to the competition, she could have won.

Catatan Penting!!!

Untuk Second Conditional, 'was' selalu digantikan 'were' untuk semua subjek. Contoh: If he were not your brother, I would not lend my motorbike to him.

4. Zero Conditional

Zero conditional digunakan untuk pengandaian yang menyatakan kenyataan yang akan terjadi secara alami, dapat dijelaskan dengan alasan ilmiah. Akibat dari pengandaian ini selalu benar.

POLA kalimat: If + Present Tense, Present Tense

Contoh:

If I drink coffee, I feel queasy. If the flowers don't get enough water, they die.

PADA SOAL

If clause reminder untuk mengingatkan sesuatu akan terjadi jika sesuatu yang lain dilakukan. If clause ini menggunakan CS (conditional sentence) Type 1.

If clause imperative, saat terjadi sesuatu, maka seseorang memberi perintah untuk melakukan sesuatu yang lain. Setelah if clause, kata kerja kalimat kedua adalah bentuk perintah.

If clause suggestion, jika terjadi sesuatu maka sesuatu yang lain kemungkinan akan terjadi, sebagai saran. If clause ini diikuti kata 'should' apda kalimat utamanya.

If clause general truth, jika sesuatu terjadi maka sesuatu yang lain pasti terjadi sebagai konsekuensinya. If clause ini menggunakan CS Type 0.

If clause to show a dream, jika terjadi sesuatu maka sesuatu yang lain dapat dicapai atau dapat dilakukan. If clause ini dapat menggunakan if clause Type 2.

33. ^!buatkan 10 contoh kalimat if clause reminder, an imperative, suggestion, general truth, dan to show a dream. dimohon yang tau bagi2 jawabannya ya


Conditional sentences dengan menggunakan if-clause bisa digunakan untuk mengungkapkan beberapa hal. Di antaranya adalah untuk pengingat (reminder), memberikan perintah (imperative), memberikan saran (suggestion), menyampaikan sesuatu yang merupakan fakta umum (general truth), atau menunjukkan impian akan masa depan (show a dream).

Berikut adalah penjelasan masing-masing disertai 10 contoh kalimat dan artinya:

If Clause sebagai Reminder (Pengingat)

Conditional sentence ini diungkapkan untuk mengingatkan seseorang akan sesuatu yang harus dilakukan jika ingin mendapatkan hasil tertentu. Ini termasuk conditional sentence type 1 yang menggunakan simple present.

If + Simple Present, Subject + Have/Has to/Must + V1

Contoh:

1. If you want to stay healthy, you must eat organic food. (jika kamu ingin tetap sehat, kamu harus makan makanan organik)

2. If she wants to be slim, she has to go to a fitness center. (jika dia ingin ramping, dia harus ke pusat kebugaran)

3. If they want to meet me, they must come before 3 pm. (jika mereka ingin bertemu saya, mereka harus datang sebelum jam 3 sore)

If Clause sebagai imperative (perintah)

Kalimat pengandaian ini digunakan untuk menyuruh seseorang melakukan sesuatu jika dia/mereka menginginkan terjadinya suatu kondisi.

If + Simple Present, V1 + Object/Adverb

Contoh:

1. If you want to finish your homework, stop playing mobile games now! (Jika ingin menyelesaikan PRmu, berhenti main game HP sekarang juga).

2. If you want to take a good picture, choose a correct setting for the camera! (Jika ingin mengambil gambar yang bagus, pillih pengaturan yang benar untuk kameranya).

3. If you want to win this match, be confident and never surrender! (Jika ingin memenangkan pertandingan ini, percaya dirilah dan jangan pernah menyerah!).

If Clause untuk Memberikan Saran (Suggestion)

Conditional sentence yang satu ini diungkapkan seseorang untuk memberikan saran akan sesuatu yang seharusnya dilakukan jika ingin mendapatkan hasil yang diinginkan.

If + Simple Present, S + Should /Had Better/ Ought to + V1

If + Simple Present, Why don't + S + V1?

Contoh:

1. If you want to get better, you should see a doctor. (Jika ingin cepat sembuh, seharusnya kamu ke dokter).

2. If you want to get up early in the morning, why don't you stop watching TV. (Jika ingin bangun lebih pagi, seharusnya kamu berhenti nonton TV).

3. If you want to speak English well, you had better join a good English course. (Jika ingin lancar bicara bahasa Inggris, seharusnya kamu ikut kursus bahasa Inggris yang bagus).

If Clause untuk mengungkapkan fakta umum (General Truth)

Merupakan kalimat pengandaian untuk menyatakan fakta atau kebenaran yang diterima secara umum.

If + Simple Present, Future

If + Simple Present, Simple Present

1. If you heat water until 100 degrees of celcius, it will boil. (Jika kamu memanaskan air sampai 100 derajat celcius, airnya akan mendidih).

2. If you throw an apple upward, it will fall downward. (Jika kamu melempar apel ke atas, apelnya akan jatuh ke bawah).

3. If the sun rises, the morning comes. (Jika matahari terbit, pagi datang).

If Clause untuk menunjukkan impian (showing a dream)

Merupakan conditional sentence untuk menunjukkan apa yang ingin dilakukan atau didapatkan seseorang di masa depan jika dia memiliki atau berhasil melakukan sesuatu.

If + Simple Present, Future

Contoh:

1. If I have 1 billion rupiahs, I will build a hotel. (Jika punya uang 1 miliar, saya akan membangun hotel).

2. If I graduate from high school, I will continue my study to Japan. (jika lulus SMA, aku akan kuliah di Jepang).

3. If I get the scholarship, I will treat you all. (Jika dapat beasiswa, akan kutraktir kalian semua).

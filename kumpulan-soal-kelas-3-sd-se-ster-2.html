Jawaban:

a. (f o g) (x) = f(g(x))

= f(x - 2)

= 3(x - 2)² + (x - 2) + 5

= 3(x² - 4x + 4) + x - 2 + 5

= 3x² - 12x + 12 + x + 3

= 3x² - 11x + 15

b. (g o f) (x) = g(f(x)

= g(3x² + x + 5)

= (3x² + x + 5) - 2

= 3x² + x + 3

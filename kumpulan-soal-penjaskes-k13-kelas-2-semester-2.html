14. 1). latihan 4.3 kelas 8 paket semester 2 K13


Bangun ruang

1) unsur limas
a) rusuk alas = PQ,QR,RS, PS
rusuk tegak = KP, KQ, KR, KS

b. bidang tegak = KPQ, KQR, KRS , KPS

c. tinggi limas = KM

2) alas limas persegi panjang, p= 16 cm,l = 12 cm
diagonalalas = d= √p²+l²= √16²+12²
d = √400
d = 20

Tinggi limas = t = 24
Panjang rusuk tegak = s= √(t² +(1/2 d)²
s = √(24²+10²) = √676 = 26 cm

Panjang Kerangka Limas = keliling alas + jumlah panjang rusuk tegak
K = 2(p+l) + 4s
K = 2(16+12) + 4(26)
K = 56 + 104 = 160
Panjang Kawat = K = 160 cm

24. Jawaban ipa halaman 36 k13 kelas 9 semester 2


Jawab :

Tegangan primer sebesar 15 Volt, dan arus sekunder sebesar 20 mA.Tegangan sekunder sebesar 11 Volt.Efesiensi sebesar 20%.

Penjelasan :

Nomer 1 :Diketahui :

Np = Lilitan primer = 1.500

Ns = Lilitan sekunder = 300

Vs = Tegangan sekunder 3 Volt

Ip  = Arus primer = 4 mA = 0,004 A

Ditanyakan :

Vp = Tegangan primerr = ...

Is   = Arus sekunder = ....

Penyelesaian :

[tex]\boxed{\frac{Np}{Ns}=\frac{Vp}{Vs}=\frac{Is}{Ip}}[/tex]

Besar tegangan primer (Vp) :

[tex]\frac{Np}{Ns}=\frac{Vp}{Vs}[/tex]

[tex]\frac{1.500}{300}=\frac{Vp}{3}[/tex]

300Vp = 3 x 1.500

[tex]Vp=\frac{4.500}{300}[/tex]

Vp = 15 Volt

Besar Arus sekunder (Is) :

[tex]\frac{Np}{Ns}=\frac{Is}{Ip}[/tex]

[tex]\frac{1.500}{300}=\frac{Is}{0,004}[/tex]

300Is = 0,004 x 1.500

300Is = 6

[tex]Is=\frac{6}{300}[/tex]

Is = 0,02 A

   = 20 mA

=======================

Nomer 2 :Diketahui :

Np = Lilitan primer = 1.200

Ns = Lilitan sekunder = 40

Vp = Tegangan primer = 330 Volt

Ditanyakan :

Vs = Tegangan sekunder = ....

Penyelesaian :

[tex]\frac{Np}{Ns}=\frac{Vp}{Vs}[/tex]

[tex]\frac{1.200}{40}=\frac{330}{Vs}[/tex]

1.200Vs = 40 x 330

1.200Vs = 13.200

[tex]Vs=\frac{13.200}{1.200}[/tex]

Vs = 11 Volt

=================

Nomer 3 :

Diketahui :

Pp = Daya primer = 350 watt

Ps = Daya sekunder = 70 watt

Ditanyakan :

η = Efesiensi = ....

Penyelesaian :

[tex]\boxed{\mu=\frac{Ps}{Pp}\times100\% }[/tex]

[tex]\mu=\frac{70}{350}\times100\%[/tex]

η = 20%

19. diketahui fungsi kuadrat yang grafiknya melalui titik A(0,4) B(-1,7) C(1,3) nilai ekstrim funsi kuadrat itu adalah


Dikarenakan melalui tiga titik sembarang, maka fungsinya dapat dinyatakan dalam:
f(x) = ax² + bx + c

Apabila disubstitusikan masing-masing dengan titik tersebut, akan terbentuk SPL tiga variabel.

Dengan melalui A(0,4):
4 = a.0² + b.0 + c
c = 4 ... (1)

Melalui B(-1,7)
7 = a.(-1)² + b.(-1) + c
7 = a - b + 4
3 = a - b
b = a - 3 ... (2)

Melalui C(1,3)
3 = a.1² + b.1 + c
3 = a + b + 4
-1 = a + b
Substitusi (2) ke persamaan berikut, menghasilkan:
-1 = a + (a-3)
-1 = 2a - 3
2a = 2
  a = 1

Diperoleh a = 1, sehingga b = a - 3 = -2, dan c = 4

Didapat: f(x) = x² - 2x + 4

Nilai ekstrim sebuah fungsi kuadrat diberikan dalam:
= -D/4a
= - (b² - 4ac) / 4a
= - ((-2)² - 4(1)(4)) / 4(1)
= - (4 - 16)/4
= 12/4
= 3

Sehingga, nilai ekstrin fungsi kuadrat tersebut adalah 3.

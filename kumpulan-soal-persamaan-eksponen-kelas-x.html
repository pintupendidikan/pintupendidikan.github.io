15. Materi Persamaan Eksponen Kelas X


Materi Persamaan Eksponen 
Matematika X

[tex]3^{2x+2}+8.3^{x}-1=0[/tex]
Misalkan 
[tex]p=3^{x}[/tex]
Maka
[tex]9.(3^{x})^{2}+8.3^{x}-1=0[/tex]
Menjadi
[tex]9.p^{2}+8.p-1=0[/tex]
Faktorkan
(9p - 1)(p + 1) = 0
Diperoleh 
[tex]p= \frac{1}{9}[/tex]
Atau
[tex]p=-1[/tex]
Selanjutnya, kembalikan p ke bentuk semula

Bagian Pertama
[tex]3^{x}= \frac{1}{9}[/tex]
[tex]3^{x}= 3^{-2}[/tex]
x = - 2

Bagian Kedua
[tex]3^{x}=-1[/tex]
Tidak ada nilai x yang memenuhi

Mengenai interval, perhatikan dengan cermat penjelasan berikut ini.
⇒ Telah diperoleh penyelesaian yakni x = -2
⇒ Interval yang tertulis di pilihan ganda adalah,
A. -¹/₂ ≤ x ≤ 0
B. -3 ≤ x ≤ -1 
C. -¹/₂ ≤ x ≤ ¹/₂
D. ¹/₂ ≤ x ≤ 1
E. 1 ≤ x ≤ 2

Jadi, posisi x = -2 berada di interval [-3, -1]

--------------------------------
Keterangan:
Tanda interval [a, b] bermakna a ≤ x ≤ b, sedangkan interval (a, b) bermakna a < x < b, jadi kalau [a, b) artinya a ≤ x < b

- Basis pangkat sama dengan 0
x² + 7x + 10 = 0
(x+2)(x + 5) = 0

- Basis pangkat sama dengan 1
x² + 7x + 10 = 1
x² + 7x + 9 = 0
Dengan rumus ABC akan didapat:
x = 1/2 (-7 + √13)
x = 1/2 (-7 - √13)

- Fungsi pangkat sama:
3x - 2 = 5x - 4
2x = 2
x = 1

HP = {x | 1/2 (-7 - √13), -5, 1/2 (-7 + √13), -2, 1}

bagian h.
- Basis sama dengan 0
x² - x + 1 = 0
(Tidak membentuk penyelesaian)

- Basis sama dengan 1
x² - x + 1 = 1
x² - x = 0
x (x - 1) = 0
x = 0 dan x = 1

- Fungsi pangkatnya sama.
2x² + x - 6 = x² - 2x + 2
x² + 3x - 8 = 0
Dengan rumus ABC:
x = 1/2 (-3+√41)

§ Eksponen §

Nomor 1

[tex] {3}^{2x - 2} = 1[/tex]

Kita ketahui bahwa :

[tex] { \alpha }^{0} = 1[/tex]

Maka nilai dari

[tex]2x - 2 = 0[/tex]

[tex]2x = 0 + 2[/tex]

[tex]2x = 2[/tex]

[tex]x = \frac{2}{2} [/tex]

[tex]{ \boxed{ \underline{x = 1}}}[/tex]

Nomor 2

[tex] {3}^{ \frac{x - 2}{3} } = 1[/tex]

Menggunakan prinsip yang sama bahwa :

[tex] { \alpha }^{0} = 1[/tex]

Maka nilai dari :

[tex] \frac{x - 2}{3} = 0 [/tex]

[tex]x - 2 = 0 \times 3[/tex]

[tex]x - 2 = 0 [/tex]

[tex]{ \boxed{ \underline{x = 2}}}[/tex]

Nomor 4

[tex]3 \sqrt{ {3}^{x - 1} } = 1[/tex]

Maka dapat diketahui bahwa

[tex] \sqrt{ {3}^{x - 1} } = 1 \div 3[/tex]

[tex] \sqrt{ {3}^{x - 1} } = \frac{1}{3} [/tex]

Untuk mengubah dari bentuk akar menjadi bentuk eksponen gunakan sifat :

[tex]{ \boxed{ \bold{ \sqrt[y]{ { \alpha }^{x} } = { \alpha }^{ \frac{x}{y} } }}}[/tex]

Maka bentuk eksponennya adalah :

[tex]{3}^{ \frac{x - 1}{2} } = \frac{1}{3} [/tex]

[tex] \frac{1}{3} [/tex] dapat kita ubah menjadi bentuk eksponen berpangkat min

[tex] \frac{x - 1}{2} = - 1[/tex]

[tex]{ \boxed{ \underline{x = - 1}}}[/tex]

37. persamaan eksponen kelas 10​


Penjelasan dengan langkah-langkah:

h(x)^f(x) = h(x)^g(x)

Syarat - syaratnya :

• f(x) = h(x)

• h(x) = 1

• h(x) = 0 (untuk f(x), g(x) > 0)

• h(x) = -1 (untuk nilai f(x) dan g(x) keduanya genap atau keduanya ganjil)

(x - 3)ˣ⁻⁶ = (x - 3)³

h(x) = x - 3, f(x) = x - 6, g(x) = 3

• untuk f(x) = g(x)

x - 6 = 3

x = 3 + 6

x = 9

• untuk h(x) = 1

x - 3 = 1

x = 1 + 3

x = 4

• untuk h(x) = 0

x - 3 = 0

x = 3

coba kita cek :

f(3) = 3 - 6 = -3 (tidak memenuhi)

karena nilai f(x) < 0, berarti x = 3 tidak memenuhi

• untuk h(x) = -1

x - 3 = -1

x = -1 + 3

x = 2

kita cek :

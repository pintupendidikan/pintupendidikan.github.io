17. Selesaikan dengan cermat soal dibawah ini berdasarkan proses pengerjaan yang terperinci.Soal:Buktikan bahwa diketahui bilangan real x ada persis bilangan n dan £ yang unik sedemikian rupa sehingga x = n + £, n adalah bilangan bulat, dan 0 ≤ £ < 1.​


Jawaban:

Terbukti bahwa ada nilai real x yang memenuhi pada sistem tersebut yaitu [tex] \sf x \in \mathbb{Z} \: , \: x \ne 0 [/tex]

Penjelasan dengan langkah-langkah:

[tex] \: [/tex]

Diketahui:

[tex]\left\{\begin{array}{l} \sf x = n + \pounds \\ \\ \sf x = n \\ \\ \sf x,n \ne \pounds \\ \\ \sf \: x,n \: \in\: \mathbb{Z} \\ \\ 0 \leqslant \pounds < 1\end{array} \right.[/tex]

[tex]\:[/tex]

Ditanya:

Buktikan bahwa ada nilai real x yang memenuhi sistem tersebut

[tex]\:[/tex]

Penyelesaian:

[tex]\:[/tex]

[tex]\begin{aligned}\sf x &\sf= n + \pounds \: \: \rightarrow \: \: x = n \\ \sf n &\sf= n + \pounds \\ \sf n - n &\sf= \pounds \\ \sf \pounds &\sf= 0 \: \: \: \rightarrow \scriptsize memenuhi \: rentang \: 0 \leqslant \pounds < 1 \end{aligned}[/tex]

Jadi, [tex] \boxed{\green{\pounds = 0}}[/tex] memenuhi

[tex]\:[/tex]

[tex]\begin{aligned} \sf x & \sf= n + \pounds \\ \sf x & \sf= n + 0 \\\sf x & \sf= n \\ \sf \sf x,n \: & \sf \in \mathbb{R}\end{aligned}[/tex]

Dimana x dan n juga bilangan bulat ([tex]\sf x,n \in \mathbb{Z}[/tex]), maka:

[tex] \sf x,n \: \in \: \{\mathbb{R}\cap\mathbb{Z}\} \\ \boxed{ \green{\sf x,n \: \in \: \mathbb{Z}}}[/tex]

[tex]\:[/tex]

[tex]\sf x,n \: \ne \: \pounds \\ \boxed{\green{\sf x,n \: \ne 0}}[/tex]

[tex]\:[/tex]

Himpunan Penyelesaian:

[tex]\sf HP : \{ ( x,n,\pounds ) \: \: | \: \: x=n \:; \: \pounds = 0 \: \: ; \: \: \underline{x,n \in \mathbb{Z}}\: \: ; \: \: \underline{x,n \: \ne 0 }\}[/tex]

18. tentukan himpunan selesaian dari pertidak samaan berikut dengan x adalah anggota himounan bilangan real . kemudian lukiskan penyelesaiannya dalam garis bilangan , soal -8 ≤ 2/5 ( k - 2 )


Diberikan pertidaksamaan -8 ≤ 2/5 (k – 2), dengan k ∈ bilangan real.
Penyelesaian pertidaksamaan di atas adalah:
-8 ≤ 2/5 (k – 2)
↔ -8 × 5/2 ≤ k – 2
↔ -20 ≤ k – 2
↔ -20 + 2 ≤ k
↔ -18 ≤ k
↔ k ≥ -18
Kita peroleh himpunan penyelesaian, Hp = {k ≥ -18, k ∈ bilangan real}.
Jika digambarkan dengan garis bilangan, maka himpunan penyelesaian tersebut akan terwakili oleh sebuah tanda panah yang bermula di titik k = -18 dengan bulatan tertutup menuju kea rah kanan dari titik k = -18.
Bulatan tertutup menunjukkan bahwa nilai k = -18 termasuk ke dalam himpunan penyelesaian.
Gambar garis bilangan terlampir.

26. cari 2 soal kalimat di bikin type1 type 2 real type 3 real


Kelas: SMA
Pelajaran: English
Kategori: conditional
Pembahasan: 

Type 1 (possible event)
- If she calls me, I'll ask her.
- If she doesn't call me, I won't call her first.

Type 2 (imagination)
- If the alien attacked earth, I'd try to get to know it.
(the real: alien doesn't attack earth, so I won't try to get to know it)
- if North Korea started the war against USA, there would be a World War III. (the real: North Korea doesn't start the war yet, so there is no WW III.)

Type 3 (regrets)
- I would have sent it earlier, if I had known about it. 
(the real: I didn't know about it, so I wouldn't send it earlier)
- The murderer would have been caught, if the cops hadn't been fooled by him. (the real: the murderer wasn't caught because the cops were fooled by him)

Cara :

[tex]3y - 1 = 17[/tex]

[tex] \frac{3k}{3} = \frac{15}{3} [/tex]

[tex]k = 5[/tex]

29. bagaimana pembuktian analisis real a.0=0


Anggap kita memiliki bilangan a dan b

Anggap teorema sudah dibuktikan bahwa untuk b bilangan real, maka berlaku :
0 + b = b

Sehingga,

a*(0 + b) = a*b
a(0) + a(b) = a(b)

Manipulasi nilai,

a(0) + a(b) - a(b) = a(b) - a(b)
a(0) = 0

Terbukti bahwa a(0) = 0
a*0 = 0

1. Salah satu faktor dari 2x³ - 5x² - px + 3 adalah ( x + 1 ). Faktor lain dari suku banyak tersebut adalah...

A. (x - 2) dan (x - 3)

B. (x + 2) dan (2x - 1)

C. (2x - 1) dan (x - 3)

D. (2x + 1) dan (x - 2)

Pembahasan :

Karena x + 1 merupakan faktor suku banyak, maka suku banyak habis dibagi dan berlaku f(-1) = 0.

f(x) = 2x² - 5x² - px + 3 dari (x + 1) diperoleh x = -1

=> f(-1) = 2(-1)³ - 5(-1)² - p(-1) + 3

=> 0 = -2 - 5 + p + 3

=> 4 = p => substitusi nilai p = 4 ke fungsi suku banyak.

f(x) = 2x³ - 5x² - 4x + 3

=> f(x) = (x + 1) (2x² - 7x + 3)

=> f(x) = (x + 1) (2x - 1) (x - 3)

Jadi faktor lainnya adalah

C. (2x - 1) dan (x - 3)

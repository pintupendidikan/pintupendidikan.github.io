25. tolong bantu ya soal ujian mandiri


ax² + 5x - 12 = 0

Salah satu akar persamaannya adalah 2, jadi:


(ax + .......) (x - 2) = 0 ⇒ salah satu akar persamaannya adalah 2 ( yaitu x-2)


Isian titik-titik tersebut adalah:


...... di kali -2 = -12 ⇒ jadi titik-titik tersebut adalah 6.


Jadi: (ax + 6) (x - 2) = 0


Untuk mencari nilai a adalah:


(ax dikali -2) + (6 dikali x) = 5x

-2ax + 6x = 5x

-2a + 6 = 5

-2a = -1

a = 1/2



Jadi ⇒ (1/2x + 6) (x - 2) = 0


Akar persamaan yang lainnya adalah:


(1/2x + 6) = 0

1/2x = -6

x = -12

benar => +4

salah => -1

Fina menjawab semua, misal :

x => banyak soal yang dijawab benar

120 = (x × 4) + ((45 - x) × -1)

120 = 4x + (-45 + x)

120 = 4x + x - 45

120 = 5x - 45

5x = 165

x = 33

jumlah soal salah = 45 - 33 = 12 buah

Jadi, jumlah soal yang dijawab salah = 12.
